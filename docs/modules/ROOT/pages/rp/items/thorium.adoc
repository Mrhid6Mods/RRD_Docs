= Thorium

Thorium is a new ore introduced by the Refined Power mod.
It is mined and processed into Thorium Fuel for use in the Modular Reactor.

== Properties and Specifications

- **Type**: Ore
- **Unlocked in**: HUB Tier 7 (Refined Power Mk4)

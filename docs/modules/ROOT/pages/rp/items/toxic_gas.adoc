= Toxic Gas

Toxic Gas can be extracted from toxic gas pillars found in the world, used to produce Methane Gas for steam generation in the Solution Heater.

== Properties and Specifications

- **Type**: gas
- **Energy Value**: Not applicable
- **Byproducts**: None
- **Unlocked in**: HUB Tier 6 (Modular Combustion)

== Usage

Toxic Gas is processed to produce Methane Gas, which is used in the Solution Heater to generate steam.
See the link:methane_gas.adoc[Methane Gas] page for details.

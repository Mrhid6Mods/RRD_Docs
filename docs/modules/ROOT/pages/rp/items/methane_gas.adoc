= Methane Gas

Methane Gas is produced from Toxic Gas, which is extracted from toxic gas pillars.
It is used in the Solution Heater to generate steam.

== Properties and Specifications

- **Type**: Gas
- **Energy Value**: Not applicable
- **Byproducts**: None
- **Unlocked in**: HUB Tier 6 (Modular Combustion)

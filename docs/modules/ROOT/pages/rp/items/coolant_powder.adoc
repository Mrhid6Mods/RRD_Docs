= Coolant Powder

The Coolant Powder is a consumable item used as a key component in producing cooling fluids for the Modular Reactor, aiding in heat management for advanced power generation.

== Properties and Specifications

- **Type**: Advanced Component
- **Energy Value**:
- **Byproducts**: None
- **Unlocked in**: HUB Tier 7 (Refined Power Mk4)

== Description

The Coolant Powder is an advanced material used in the production of Reactor Coolant, essential for maintaining stable operation of the Modular Reactor.
It plays a critical role in managing heat levels, preventing emergency shutdowns in high-output power setups.

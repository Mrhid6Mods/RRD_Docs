= Micro Reactor

== Building Info

The *Micro Reactor* is a single-building nuclear generator that outputs *1{nbsp}GW* of power *without producing any waste*! Unlike the multi-part xref:/rp/buildings/modularreactor/index.adoc[Modular Reactor], there is nothing to assemble, you place one building, load it with fuel, and it feeds the grid.

In-game description: _"A small micro reactor capiable of 1GW of power!"_

== Building Cost

5 Supercomputers, 40 Alclad Aluminium sheets & 30 A.I. Limiters

== Consumption Rates
The Micro Reactor can be fed with the following items:

|===
| Item   | Consumption rate per minute
| Nuclear Fuel Rods|0.08 
| Plutonium|0.04 
| DT Reactor Cores|0.2
| Deanorium Reactor Cores|0.05
| Ficsonium Fuel Rods|0.4
|===
